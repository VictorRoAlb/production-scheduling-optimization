# Public Example Instance

This folder contains a safe CSV export of one executed benchmark instance from the PPP project.

Included files:

- `instance_csv/release_dates.csv`
- `instance_csv/due_dates.csv`
- `instance_csv/pesos.csv`
- `instance_csv/precedencias.csv`
- `instance_csv/recursos.csv`
- `instance_csv/setups_M1.csv` to `instance_csv/setups_M5.csv`

These files document the structure of the benchmark without publishing the original Excel workbook.

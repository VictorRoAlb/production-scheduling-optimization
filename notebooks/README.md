# Public notebooks

This folder keeps notebook versions of the two main execution stages of the project:

- `01_instance_generation_from_workbook.ipynb`
  Expands the base processing-time workbook into release dates, setup times, due dates, weights, precedences and resource-capacity parameters.
- `02_benchmark_extensions_execution.ipynb`
  Mirrors the main solving workflow with the constructive evaluator, CP-SAT, the genetic algorithm, iterated greedy and the six benchmark extensions.

The original workbook is not included in the public repository, so the notebooks are meant to document the real execution logic rather than serve as one-click runnable artifacts out of the box.
